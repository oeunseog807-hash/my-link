# 마이링크 (My Link) - 사용자 시나리오

## 1. 방문자 시나리오

> 마이링크 공개 페이지를 방문하는 사용자의 흐름

### 1-1. 공개 페이지 방문

1. 방문자는 **마이링크 페이지를 보기 위해** 공유받은 URL(`/{displayName}`)에 접속한다.
2. 방문자는 **페이지 소유자가 누구인지 확인하기 위해** 프로필 이미지, username, 소개글을 확인한다.
3. 방문자는 **원하는 링크로 이동하기 위해** 링크 목록에서 파비콘과 제목을 보고 원하는 링크를 클릭한다. (클릭 수 카운트)
4. 방문자는 **해당 링크의 외부 페이지로 이동하기 위해** 새 탭에서 열린 페이지를 확인한다.

### 1-2. 랜딩 페이지 방문

1. 방문자는 **마이링크 서비스가 무엇인지 알기 위해** 랜딩 페이지(`/`)에 접속한다.
2. 방문자는 **자신도 마이링크를 만들기 위해** 로그인 버튼을 클릭한다.

---

## 2. 소유자 시나리오

> 마이링크 페이지를 소유하고 관리하는 사용자의 흐름

### 2-1. 회원가입 (최초 로그인)

1. 소유자는 **마이링크를 시작하기 위해** 로그인 페이지(`/login`)에서 Google 로그인 버튼을 클릭한다.
2. 소유자는 **Google 계정을 연동하기 위해** Google 인증 팝업에서 계정을 선택한다.
3. 소유자는 **프로필을 설정하기 위해** 설정 페이지(`/setup`)에서 username을 입력한다.
4. 소유자는 **공개 페이지 URL을 확인하기 위해** Gmail 앞부분으로 자동 생성된 displayName을 확인한다.
5. 소유자는 **가입을 완료하기 위해** 설정을 저장하고 관리 대시보드로 이동한다.

### 2-2. 링크 추가

1. 소유자는 **새 링크를 등록하기 위해** 관리 대시보드(`/admin`)에서 링크 추가 버튼을 클릭한다.
2. 소유자는 **링크 정보를 입력하기 위해** 제목과 URL을 작성한다.
3. 소유자는 **링크를 저장하기 위해** 저장 버튼을 클릭한다.
4. 소유자는 **링크가 정상 등록되었는지 확인하기 위해** 링크 목록에서 파비콘과 함께 추가된 링크를 확인한다.

### 2-3. 링크 수정

1. 소유자는 **기존 링크를 수정하기 위해** 관리 대시보드에서 수정할 링크를 클릭한다.
2. 소유자는 **링크 정보를 변경하기 위해** 인라인 편집으로 제목 또는 URL을 수정한다.
3. 소유자는 **변경 사항을 저장하기 위해** 편집을 완료한다.

### 2-4. 링크 삭제

1. 소유자는 **필요 없는 링크를 제거하기 위해** 관리 대시보드에서 삭제할 링크의 삭제 버튼을 클릭한다.
2. 소유자는 **실수로 삭제하는 것을 방지하기 위해** 확인 다이얼로그에서 삭제를 확정한다.
3. 소유자는 **삭제가 반영되었는지 확인하기 위해** 링크 목록에서 해당 링크가 사라진 것을 확인한다.

### 2-5. 프로필 수정

1. 소유자는 **프로필 정보를 변경하기 위해** 관리 대시보드에서 프로필 편집 영역으로 이동한다.
2. 소유자는 **이름을 변경하기 위해** username을 수정한다.
3. 소유자는 **공개 URL을 변경하기 위해** displayName을 수정한다.
4. 소유자는 **자기소개를 변경하기 위해** 소개글을 수정한다.
5. 소유자는 **변경 사항을 적용하기 위해** 저장 버튼을 클릭한다.
