# 마이링크 (My Link) - 와이어프레임

## 페이지 흐름 (Mermaid)

```mermaid
flowchart TD
    A[/ 랜딩 페이지] -->|로그인 클릭| B[/login 로그인]
    B -->|Google 인증| C{최초 가입?}
    C -->|Yes| D[/setup 초기 설정]
    C -->|No| E[/admin 관리 대시보드]
    D -->|설정 완료| E
    E -->|공개 페이지 보기| F[/displayName 공개 페이지]
    E -->|로그아웃| A
    F -->|링크 클릭| G[외부 URL]
```

---

## 1. 랜딩 페이지 (`/`)

```
+------------------------------------------+
|  [로고] 마이링크          [로그인 버튼]   |
+------------------------------------------+
|                                          |
|                                          |
|         나만의 링크 페이지를              |
|         1분 만에 만들어보세요             |
|                                          |
|         [시작하기 버튼]                   |
|                                          |
|                                          |
+------------------------------------------+
```

## 3. 초기 설정 페이지 (`/setup`)

```
+------------------------------------------+
|              프로필 설정                  |
+------------------------------------------+
|                                          |
|    +--------+                            |
|    | Google |  프로필 이미지 (자동)       |
|    |  사진  |                            |
|    +--------+                            |
|                                          |
|    username                              |
|    +----------------------------------+  |
|    | 홍길동                           |  |
|    +----------------------------------+  |
|                                          |
|    displayName (URL)                     |
|    +----------------------------------+  |
|    | gildong123  (Gmail에서 자동 설정)|  |
|    +----------------------------------+  |
|    미리보기: mylink.site/gildong123      |
|                                          |
|    소개글                                |
|    +----------------------------------+  |
|    | 안녕하세요! 개발자 홍길동입니다. |  |
|    +----------------------------------+  |
|                                          |
|         [저장하고 시작하기]              |
|                                          |
+------------------------------------------+
```

## 4. 관리 대시보드 (`/admin`)

```
+------------------------------------------+
|  [로고] 마이링크     [공개페이지] [로그아웃]|
+------------------------------------------+
|                                          |
|  +-- 프로필 ---------------------------+ |
|  |                                     | |
|  |  +--------+                         | |
|  |  | Google |  username: 홍길동 [편집]| |
|  |  |  사진  |  @gildong123      [편집]| |
|  |  +--------+  소개글: 안녕하세.. [편집]| |
|  |                                     | |
|  +-------------------------------------+ |
|                                          |
|  +-- 링크 목록 ------------------------+ |
|  |                                     | |
|  |  +-------------------------------+  | |
|  |  | [favicon] 내 블로그      [삭제]|  | |
|  |  |   https://blog.example.com    |  | |
|  |  +-------------------------------+  | |
|  |                                     | |
|  |  +-------------------------------+  | |
|  |  | [favicon] GitHub         [삭제]|  | |
|  |  |   https://github.com/user     |  | |
|  |  +-------------------------------+  | |
|  |                                     | |
|  |  +-------------------------------+  | |
|  |  | [favicon] 포트폴리오     [삭제]|  | |
|  |  |   https://portfolio.dev       |  | |
|  |  +-------------------------------+  | |
|  |                                     | |
|  |         [+ 링크 추가]              | |
|  |                                     | |
|  +-------------------------------------+ |
|                                          |
+------------------------------------------+
```

### 링크 추가 시

```
+------------------------------------------+
|  +-- 새 링크 추가 ---------------------+ |
|  |                                     | |
|  |  제목                               | |
|  |  +-------------------------------+  | |
|  |  |                               |  | |
|  |  +-------------------------------+  | |
|  |                                     | |
|  |  URL                               | |
|  |  +-------------------------------+  | |
|  |  | https://                      |  | |
|  |  +-------------------------------+  | |
|  |                                     | |
|  |         [취소]  [저장]              | |
|  |                                     | |
|  +-------------------------------------+ |
+------------------------------------------+
```

### 링크 인라인 편집 시

```
+----------------------------------------------+
|  +----------------------------------------+  |
|  | [favicon] [내 블로그        ] [저장][취소]|  |
|  |   [https://blog.example.com ]          |  |
|  +----------------------------------------+  |
+----------------------------------------------+
```

### 링크 삭제 확인

```
+------------------------------------------+
|                                          |
|    +------------------------------+      |
|    |  정말 삭제하시겠습니까?      |      |
|    |                              |      |
|    |  "내 블로그" 링크가          |      |
|    |  삭제됩니다.                 |      |
|    |                              |      |
|    |      [취소]  [삭제]          |      |
|    +------------------------------+      |
|                                          |
+------------------------------------------+
```
